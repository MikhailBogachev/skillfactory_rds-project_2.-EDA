Краткое описание

Основные цели и задачи проекта:
Суть проекта — отследить влияние условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, чтобы на ранней стадии выявлять студентов, находящихся в группе риска.
Задачи:
- Произвести первичную обработку данных
- Устранить выбросы, если таковые имеются
- Для численных перменных произвести корреляционный анализ, устранить те, которые не влияют на предсказываемую величину
- Проанализировать номинативные переменные, устранить те, которые не влияют на предсказываемую величину

Информация о данных:
Датасет с информацией о 395 учениках с 30 различными признаками. 

Этапы работы:
- первичная обработка данных
- рассмотреть все столбцы по отдельности. Для каждого посчитать кол-во уникальных значений, кол-во пропусков
- для каждого признака избавиться от пропусков и выбросов
- проанализировать численные признаки с помощью корреляционного анализа
- проанализировать номинативные признаки (box plot, тест Стьюдента)
- сформировать датасет для будущего построения модели, добавив только те признаки, которые влияют на целевую переменную

Роль заключалась предобработке данных, анализе признаков

3. Что не получилось сделать так, как хотелось? Над чем ещё стоит поработать?
Некоторые строки с выбросами и пропусками можно было не просто удалять, а заполнять обоснованными значениями. Но на это нужно потратить достаточно много времени.

4. Что интересного и полезного вы узнали в этом модуле?
Попробовал на практике применить приемы для анализа номинативных признаков.
